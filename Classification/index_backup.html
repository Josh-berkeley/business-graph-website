Classification APIs

Integrate our 18-dimensional classification system directly into your investment workflows with real-time updates and flexible querying capabilities.

GET /classify/{company_id}

Classify any private company across all 18 dimensions with confidence scores and evidence sources.

{ "company_id": "dataflow_inc", "classification": { "business_model": "platform_saas", "customer_segments": ["enterprise", "smb"], "revenue_patterns": ["subscription", "usage"], "confidence": 0.94 } }
POST /universe/generate

Generate investment universes based on multi-dimensional criteria with precision filtering.

{ "criteria": { "business_model": ["b2b_saas"], "customer_segments": ["enterprise"], "revenue_patterns": ["subscription"] }, "universe_size": 2847 }

Discover true business model peers with similarity scoring across multiple dimensions.
